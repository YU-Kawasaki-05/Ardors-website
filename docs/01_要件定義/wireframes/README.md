## 更新ルール
- 新規画面を追加した場合は、対応する `SCR-*` を必ず本ディレクトリ内のいずれかに追記する。
- 画面IDや遷移先を変更した場合は `04_画面遷移図_screen-transition.md` と同時更新する。
- 主要CTA、主要導線、関連 `FR-*` を合わせて更新し、要件ズレを防ぐ。
