# SCR-01-06 Public Core Wireframes

## SCR-01 トップ
- **目的**: Ardorsの提供価値を最短で伝え、次行動へ誘導する。
- **主要ブロック**: ヒーロー、入口分岐、信頼ブロック、サービス要約、実績ハイライト、CTA。
- **主CTA**: `相談する`（SCR-07）、`実績を見る`（SCR-04）。
- **関連要件**: FR-01, FR-07, FR-09, FR-10。

## SCR-02 サービス紹介
- **目的**: 提供内容・進め方・相談対象を明確化する。
- **主要ブロック**: サービス一覧、対応範囲、進行フロー、FAQ、CTA。
- **主CTA**: `相談する`（SCR-07）。
- **関連要件**: FR-02, FR-07, FR-09。

## SCR-03 プロフィール/ポートフォリオ
- **目的**: あなたの経験・能力・実績の信頼性を提示する。
- **主要ブロック**: プロフィール、スキル、実績サマリ、GitHub導線、CTA。
- **主CTA**: `実績を見る`（SCR-04）、`相談する`（SCR-07）。
- **関連要件**: FR-03, FR-10, FR-31。

## SCR-04 実績一覧
- **目的**: 実績を成果軸で比較し、詳細閲覧を促進する。
- **主要ブロック**: 絞り込みUI、実績カード一覧、ページネーション、CTA。
- **主CTA**: 実績カードクリック（SCR-05）、`相談する`（SCR-07）。
- **関連要件**: FR-04, FR-09。

## SCR-05 実績詳細
- **目的**: 実績の課題/対応/成果を構造化して提示する。
- **主要ブロック**: タイトル、課題、対応、成果、関連実績、次ページ推薦、CTA。
- **主CTA**: `相談する`（SCR-07）、`一覧に戻る`（SCR-04）。
- **関連要件**: FR-05, FR-09。

## SCR-06 SaaS構想紹介
- **目的**: SaaS構想の背景と今後の展望を示し、関心者を誘導する。
- **主要ブロック**: 構想説明、現状ステータス、関連リンク、CTA。
- **主CTA**: `相談する`（SCR-07）。
- **関連要件**: FR-06。
