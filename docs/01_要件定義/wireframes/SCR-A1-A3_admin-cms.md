# SCR-A1-A3 Admin CMS Wireframes

## SCR-A1 CMSログイン（P1）
- **目的**: 管理画面アクセスを運営者に限定する。
- **主要ブロック**: ログインフォーム、認証エラーメッセージ、ログインボタン。
- **主CTA**: `ログイン`（SCR-A2）。
- **関連要件**: FR-42。

## SCR-A2 CMS事例一覧（P1）
- **目的**: 事例の一覧確認と編集導線を提供する。
- **主要ブロック**: 事例テーブル、公開状態、検索/並び替え、新規作成ボタン。
- **主CTA**: `新規作成`（SCR-A3）、`編集`（SCR-A3）。
- **関連要件**: FR-40, FR-41。

## SCR-A3 CMS事例編集（P1）
- **目的**: 事例の作成/編集/削除と公開状態制御を行う。
- **主要ブロック**: 入力フォーム、下書き保存、公開切替、削除ボタン。
- **主CTA**: `保存`（SCR-A2）、`公開`（SCR-A2）。
- **関連要件**: FR-40, FR-41。
